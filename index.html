  • Tile Editor

    I've always wanted to create my own virtual worlds to explore. One tool that could help tremendously with world-building is a tile editor. A tile editor allows a level designer to sketch out levels quickly with an intuitive UI. Below is my (unfinished) tile editor created for my dungeon crawler. Currently, you can paint the ground tiles of a level, press "Save," and load the level (serialized as JSON) into the dungeon scene. Pretty neat!
